# 后续演进

## 数据量达到 100 条公开内容前

- 将当前同步内容安全检测迁移为异步审核回调，并保持异常时不公开
- 增加管理员审核页面和审核操作日志
- 增加举报、屏蔽、用户协议和隐私指引
- 增加图片服务端重新编码，替代当前的元数据拒绝策略

## 数据量达到 1,000 条前

- 将搜索从“最近 100 条内存过滤”迁移到专用搜索服务或标签倒排集合
- 将排行榜改成定时聚合，避免每次首页请求重新统计
- 使用游标分页替代数字偏移
- 增加更细粒度的异常行为检测、资源告警和费用上限
